# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-28

### Added

- Initial release of pybumpchart
- Core `bumpchart()` function for creating bump chart visualizations
- Data validation and preparation utilities
- Automatic rank calculation from values with multiple tie-breaking methods
- Color palette support (matplotlib colormaps, seaborn palettes, custom colors)
- Entity highlighting with dimming of non-highlighted entities
- Endpoint labels with collision avoidance
- Customizable styling (line width, marker size, fonts, grid)
- Full type hints and mypy strict mode compliance
- Comprehensive test suite
